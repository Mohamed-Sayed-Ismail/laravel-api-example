Laravel Location API

Command Line Interface (CLI) Only REST API

CLI-Only Access Restriction

This API is specifically designed for command-line tools and API clients.

Web browser access is intentionally blocked.

Git Repository

Clone the project to get started:

git clone https://github.com/Mohamed-Sayed-Ismail/-laravel-api-example.git

About This API

This is a specialized REST API for searching locations in Belgium by proximity and category. The API is configured to only respond to requests from command-line tools and API testing applications.

curl

Command Line Tool

Postman

Installation Guide

1 Prerequisites

Ensure you have the following installed:

PHP 8.1+ Composer Git SQLite

2 Clone Repository

git clone https://github.com/Mohamed-Sayed-Ismail/-laravel-api-example.git
cd laravel-api-example

3 Install Dependencies

composer install

4 Environment Setup

cp .env.example .env
php artisan key:generate

5 Database Setup

touch database/database.sqlite
php artisan migrate
php artisan db:seed

6 Start Server

php artisan serve

Server will start at: http://localhost:8000

Quick Test

After installation, test the API immediately using curl:

# Search for fun activities near Brussels
curl "http://localhost:8000/api/locations/search?latitude=50.8503&longitude=4.3517&search_type=fun"

# Search for medical facilities with custom distance
curl "http://localhost:8000/api/locations/search?latitude=50.8503&longitude=4.3517&search_type=medic&distance=15"

# Get all available location types
curl "http://localhost:8000/api/locations/types"

API Endpoints

GET /api/locations/search

Search locations by type within distance

Params: latitude, longitude, search_type, distance(optional)

GET /api/locations/types

Get all available location types

GET /api/locations/type/{type}

Get all locations of specific type

Available Categories

fun theater music medic restaurant bar museum park

Usage Examples

Using curl:

curl -s "http://localhost:8000/api/locations/search?latitude=50.8503&longitude=4.3517&search_type=music"

Using PHP:

$response = file_get_contents(
  "http://localhost:8000/api/locations/search?latitude=50.8503&longitude=4.3517&search_type=fun"
);

Using Python:

import requests
response = requests.get(
  "http://localhost:8000/api/locations/search",
  params={"latitude": 50.8503, "longitude": 4.3517, "search_type": "medic"}
)

Access Notes

This API will return 403 Forbidden error when accessed from:

  • Web browsers (Chrome, Firefox, Safari, Edge)
  • Any request with browser User-Agent headers

Allowed access from:

  • Command line tools (curl, wget, httpie)
  • API clients (Postman, Insomnia, Thunder Client)
  • Custom scripts and applications
  • Laravel Artisan commands

Troubleshooting

Common Issues:

  • PHP not found: Install PHP or add to PATH
  • Composer not found: Download from getcomposer.org
  • SQLite issues: Install php-sqlite3 package
  • Port busy: Use php artisan serve --port=8001
  • Permission errors: Run chmod -R 755 storage bootstrap/cache

Verification Commands:

# Check PHP version
php --version

# Check Composer
composer --version

# Check SQLite
php -m | grep sqlite
